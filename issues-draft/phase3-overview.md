# [Phase3 概要] インゲームアイテム + ゴールド + ポーズ中購入

> **NOTE**: 概要のみ。Phase 2 完了後に詳細 Issue へ展開する。

## 背景

[`docs/roadmap.md`](../docs/roadmap.md) Phase 3 のスコープ: インゲームアイテム (素材→完成アイテムビルド) とゴールド経済を導入する。

## 含まれる主なテーマ

- ゴールド (試合中通貨) の追加
- ブロック破壊でのゴールド獲得
- インゲームアイテム
  - 素材アイテム
  - 完成アイテム (素材の組合せでビルド)
  - パッシブ効果付与
- ポーション (HP回復、シールド、短時間バフ)
- ポーズ画面と購入 UI
- アイテムによるステータスとパッシブの計算統合
- リプレイへの影響 (購入操作も再生対象)

## 留意点

- アイテムによる状態の複雑化に伴い、`World` の構造とテストケースが膨らむ
- HUD のアイテム枠表示
- ポーションの効果時間管理 (バフ機構と統合)
