# [Phase1][H-06] スキル2: スリング強化 (サイズ拡大 / チャージ倍率上昇 / 貫通)

## 背景

スキル2は「一時的にスリングを強化する」アクティブスキル ([`docs/game-design.md`](../docs/game-design.md) §3.2)。3つの効果を併せ持ち、組み合わせの動作確認をしやすい構成。

## ゴール

- スキル発動で一定時間、以下が同時に有効化される:
  - バーサイズ拡大 (`BarState.width *= multiplier`)
  - チャージ倍率上昇 (`character.stats.chargeShotMultiplier` にバフ乗算)
  - ボール貫通 (バフ期間中にスリング射出されたボールはブロックを N 個まで貫通)

## 作業内容

1. `src/core/skills/slingBuff.ts` に `SkillDef` を定義する。
   - `onActivate`: バーサイズ・チャージ倍率にバフを付与、活性タイマーをセット
2. `BallState` に `pierceRemaining?: number` を追加し、貫通残数を管理する。スリング射出時にバフが活性化していれば付与する。
3. `blockCollision` (C-06) で `pierceRemaining` を消費するロジックを追加する (1 を超える間はボール速度を反転せず通過、HP は減らす、回数で停止)。
4. テスト:
   - スキル発動でバーサイズが大きくなる
   - チャージ倍率がバフ期間だけ高くなる
   - 期間中に射出されたボールが貫通する
   - 期間終了で全効果が解除される

## 受け入れ条件

- [ ] テストが緑。
- [ ] 実機でスキル2発動 → バー大型化・チャージ倍率上昇・貫通の3効果が見える。

## スコープ外

- バーサイズの動的反射計算の最適化 (現状の C-05 の式で対応できる前提)
