# [Phase1][U-05] リプレイ再生

## 背景

リプレイファイル (F-13) を読み込んで再生できるようにする。マッチヒストリー (U-04) からの再生と、タイトルからの「リプレイファイルを読み込む」操作の両方をサポートする。

## ゴール

- リプレイファイルを読み込んで、保存された試合と同一の挙動を再現する。
- 再生中であることが UI で明示される (HUD に「REPLAY」表示など)。
- 再生中は入力を無視する。

## 作業内容

1. `src/ui/scenes/ReplayScene.ts` を作成 (または GameScene にモードを追加)。
2. リプレイ読み込み:
   - タイトル画面に「リプレイ読み込み」ボタンを追加 (ファイル選択ダイアログ → JSON 読み込み)
   - マッチヒストリー画面の各エントリに「再生」ボタンを追加
3. World 構築時に `ReplayInputSource` (F-13) を入力源として注入する。
4. ボーナス: 再生速度 (1x / 2x / 4x) の切替を実装すると望ましい (本 Issue では 1x のみ必須)。
5. テスト:
   - 同一リプレイで同一の最終 World 状態に到達する (E2E より単体推奨)
   - 再生中はマウス入力が無視される

## 受け入れ条件

- [ ] 実機でリプレイの再生ができる。
- [ ] テストが緑。

## スコープ外

- 早送り・巻き戻し (将来)
- リプレイ編集
