# [Phase1][H-07] アルティメット: ボール複製

## 背景

アルティメットは「ボール複製」 ([`docs/game-design.md`](../docs/game-design.md) §3.2)。一定時間 or 永続でボールを増やす。

## ゴール

- アルティメット発動でボールが N 個複製される。
- 複製ボールは元ボールの位置から少しずつ角度をずらして発射される。
- 複製数 N、持続方式 (時間経過で消えるか、敵を倒すまで残るか) は仕様確定して実装する。
  - 推奨初期仕様: **時間制限あり (例: 10秒)。期間中に新規生成された複製は順次消滅する。** 元ボール (id 由来の主ボール) は消えない [要調整]

## 作業内容

1. `src/core/skills/ultBallClone.ts` に `SkillDef` を定義する。
   - `onActivate`: 既存ボールごとに `N - 1` 個の複製を生成 (合計 N 倍)
   - 複製は寿命付きで、`BallState.lifetimeMsRemaining?` を持つ
2. `src/core/systems/ballLifetime.ts` を作成し、寿命付きボールの除去を行う。
3. テスト:
   - 発動でボール数が増える
   - 寿命終了で複製ボールが消える
   - 元のボールは消えない

## 受け入れ条件

- [ ] テストが緑。
- [ ] 実機でアルティメット発動すると一時的にボールが増える。

## スコープ外

- アイテムによる永続複製 (Phase 2)
