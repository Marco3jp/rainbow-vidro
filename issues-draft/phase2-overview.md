# [Phase2 概要] アイテムブロックの導入

> **NOTE**: 概要のみ。Phase 1 完了後に詳細 Issue へ展開する。

## 背景

[`docs/roadmap.md`](../docs/roadmap.md) Phase 2 のスコープ: アイテムブロック (ボール複製、貫通、強化など) を導入する。

## 含まれる主なテーマ

- アイテムブロック種別 (`itemBlock`) の追加
- ブロック破壊時のアイテムドロップ
- ピックアップロジック (バー or ボールが触れる)
- アイテム効果:
  - ボール複製 (一時 / 永続)
  - 貫通付与
  - 強化 (ダメージ・速度等のバフ)
- HUD でのアクティブアイテム表示
- リプレイ・テストへの影響評価

## Phase 1 完了後にやること

- 上記テーマを各 1 PR で完結する Issue に分割
- アイテム種別ごとのデザインドキュメント追加
- `docs/game-design.md` の §6 「1stバージョン以降の追加要素」に詳細を反映
