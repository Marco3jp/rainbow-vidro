# [Phase1][H-09] スキルレベル振り分け UI

## 背景

レベルアップで得たスキルポイントを、スキル / アルティ / パッシブのいずれかに振り分ける ([`docs/game-design.md`](../docs/game-design.md) §3.3)。

## ゴール

- スキルポイントがあるとき、UI 上で振り分け先を選択できる。
- 振り分けによってスキルレベルが上がり、効果が強化される。
- スキルレベルによる効果の差はデータ化されている。

## 作業内容

1. `CharacterState.skillLevels` を活用し、各スキルが現在何レベルかを管理する。
2. `src/core/systems/skillPoints.ts` を作成する。
   - `assignSkillPoint(character, skillId)`: スキルポイント消費 + skillLevel++
   - 上限 (例: アクティブは 5、アルティは 3) [要調整]
3. UI 側 (HUD or レベルアップモーダル) でボタンを表示し、押下で `assignSkillPoint` を呼ぶ (UI 実装は U-02 と連動)。
4. SkillDef の各種数値はスキルレベルに応じて変動するように `levelScaling` を定義 (例: ダメージ倍率、CD など)。
5. テスト:
   - スキルポイントが正しく消費される
   - 上限を超えて振れない
   - スキル効果値がレベルに応じて変わる

## 受け入れ条件

- [ ] テストが緑。
- [ ] 実機でスキルポイント振り分け → スキル効果が強化される。

## スコープ外

- レベルアップ演出
- 振り分けの巻き戻し (将来検討)
