## Phase 1 (1stバージョン) 詳細

Issue 化対象。各 Issue は独立したコーディングエージェントが担当する想定。各項目は1 PR でビルド可能な単位。

### 1.1 基盤 (Foundation)

| ID | タイトル | 概要 |
| --- | --- | --- |
| F-01 | プロジェクトセットアップ | Vite + TypeScript + npm + Biome の初期化 |
| F-02 | テスト基盤 | Vitest 導入、テストディレクトリ構成、サンプルテスト |
| F-03 | E2Eテスト基盤 | Playwright 導入、最小 E2E |
| F-04 | CI 構築 | GitHub Actions: lint + test + build |
| F-05 | GitHub Pages デプロイ | main マージで本番デプロイ |
| F-06 | PR プレビュー環境 | PR ごとにサブディレクトリへプレビュー配信 |
| F-07 | レンダラ抽象 (Pixi 初期化) | Renderer インタフェース + PixiRenderer 実装 |
| F-08 | 固定タイムステップ + 補間ループ | 60Hz シミュ + RAF 描画 + 補間 |
| F-09 | Seeded RNG | Mulberry32 と RNG 抽象 |
| F-10 | Clock 抽象 | SimClock / 実時刻 Clock |
| F-11 | Logger / エラーモニタ | console + localStorage + ダウンロード |
| F-12 | 入力抽象 (MouseInputSource) | DOM 入力 → InputEvent 化 |
| F-13 | リプレイ基盤 | 入力ログ記録・保存・再生 |
